# sr-proto
SR Protobufs & CmdId

Used for https://github.com/f2pqingque/sr

NOTE: You will need protoc to compile this.

Current version: 3.0.0

If you want the list of protos needed for a bare minimum PS, here they are:

- PlayerGetToken
- PlayerLogin
- PlayerLoginFinish
- PlayerHeartBeat
- GetBasicInfo
- GetAvatarData
- GetBag
- GetMissionStatus
- GetCurLineupData
- GetCurSceneInfo
